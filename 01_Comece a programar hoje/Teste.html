Instruções:


Tag "br" quebra linhas


Tag "h1" criar campo de texto com hierarquia e tamanhos diferentes


Tag "a" com atributo href cria link


Exemplo link - Alura

Tag "script" faz com que navegador leia o código como javascript dentro da página html


Código "alert" cria alerta em javascript, texto deve ser escrito entre aspas duplas


Tag "meta" com atributo charset="UTF-8" corrige problemas de leitura de textos com acentuação indicando ao navegador que a codificação do texto é UTF-8



Efetuado erro proposital no código javascript, para fazer a verificação do erro no navegador através da tecla F12 na aba Console
